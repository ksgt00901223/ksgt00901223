# vue 组件热更新无效

描述： vue 组件热更新无效，需重启才行。
解决： 查看组件引用路径是否正确，尤其大小写。

# vue 的 this.$router.go(0)在 safari 上无效

描述： vue 的 this.$router.go(0)在safari上无效。
解决： 用this.$router.push()或者用原生的 history.go(-1) ，location.reload() 。
