### mac 权限问题，每次都要 sudo npm

**描述：** 每次运行 npm 时都要加上 sudo，然后还要输密码。  
**解决：** 为当前账户添加 node_modules 目录读写权限。

```
sudo chown -R $(whoami) ~/.npm
```
