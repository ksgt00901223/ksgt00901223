### 打包时出现 localStorage 未定义等

**描述：** 打包时出现 localStorage、sessionStorage 等未定义报错。  
**解决：** 看运行环境是否包含 localStorage、sessionStorage，比如是否要加 window 对象，或者 ssr 打包时生命周期开始没有浏览器对象 localStorage、sessionStorage，所以会报错。需放到 mounted 钩子里。
